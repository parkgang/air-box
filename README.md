`무인택배함 관리 클라이언트 프로그램`

대학교의 스마일 박스 택배함을 참고하여 진행된 프로젝트입니다. 무인 택배함을 사용/관리하도록 고객들의 개인 정보를 기반으로 택배 수거/보관, 고객 정보와 택배기사 정보로 이용료 계산 정보 및 사용 중인 택배함을 GUI로 표현하는 프로그램입니다.

</div>

## Stack

1. Qt
1. C++

## Quick start

Qt IDE에서 실행해야 합니다.

## Issue

1. 인코딩 설정이 `euc-kr` 으로 주의합니다.
1. 각종 이미지의 경우 저작권 때문에 모두 삭제되어 있습니다.
1. `payment_window.ui` 등 개인 정보와 관련 있을 거 같은 데이터는 모두 알아볼 수 없도록 처리 및 더미 데이터로 대치되었습니다.
